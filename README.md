- IDE : IntelliJ IDEA (Community Edition)
- Programming Language : Java 17, JavaScript
- Backend Framework : `Spring Boot 3.3.1`
- Frontend Framework : `Nuxt`
- Database Access Technology : `JPA`
- Database : MySQL 8.0.34
- Version Control: Git
